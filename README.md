# Days Calculator

Days Calculator is a very very simple website tool that helps you find the number of days between two dates.

You can select a start date and end date... and the website will calculate the total days difference instantly.

## Features

- Calculate days between dates
- Include end date option
- Weeks and months and years
- Show weekday names
- Quick date 
- history 

## Languages Used

- HTML
- CSS
- Javascript

## How To Use

1. Select a start date
2. Select an end date
3. Click the calculate button
4. View the result
